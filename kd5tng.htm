An Amazing Find For All Hams

There is an excellent program on the WWW that will soon be on the computers of all Hams.  And it’s absolutely free courtesy of Simon Brown – HB9DRV

The name of this wonderful software is Ham Radio Deluxe.  And it is structured so that it can support most HF / VHF / UHF amateur radio transceivers and receivers. It currently supports ICOM, Yaesu, Elecraft, FlexRadio, Ten-Tec and Kenwood. A full list is available in the user community.  You can download Ham Radio Deluxe by pointing your browser to http://hrd.ham-radio.ch/downloads.html Be sure to download Version 3.4 to get the latest release.

Key Benefits

·    Built-in logbook,

·    Favorites (just like Internet Explorer),

·    Integrated DX cluster,
